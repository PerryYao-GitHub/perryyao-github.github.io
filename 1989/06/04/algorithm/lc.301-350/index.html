319. Bulb Switcher

Java

考虑第n行中第i个灯(1 <= i <= n), 如果存在d<i并且d|i, 则(i/d)|i, 如果d==(i/d), 则第i位置得以保留, 不然, d和(i/d)成对出现, i会熄灭

于是答案即为1…n中的完全平方数的个数
