⚙️ 三、常用命令说明

命令 说明
npm install / yarn install 安装项目依赖
npm run dev / yarn dev 启动开发服务器
npm run build / yarn build 构建静态生产文件
npm run preview 本地预览构建结果

🧠 五、注意事项

  • 默认端口是 5173，如被占用可修改 vite.config.ts

  • React 推荐使用 vite 替代 create-react-app，启动速度快很多
