DP - Subarray & Subsequence

Posted on 2024-10-21 Edited on 2024-10-25 In tech

子序列类型

Subarray - 连续子序列

Subsequence - 非连续但是保序的子序列

Subset - 不连续不保序 (集合本无顺序)

两类 dp 的定义

定义结尾 (固定结尾) - dp 本身不包含问题的答案, 而是在外部使用一个res接收dp数组中的最大值作为答案

  • 以 i结尾 的最长递增 Subsequence 的长度
  • 以 i结尾 的最长递增 Subarray 的长度

定义范围 (不固定结尾) - dp 本身包含问题答案, dp 数组的最后一项就是答案

  • text1长度为i的子串和text2长度为j的子串中, Longest Common Subsequence

另一种 dp 设法

这里只需要考虑删除或添加的一种即可, 因为仅通过 “删除 + 替换” 和 仅通过 “添加 + 替换” 两种策略 (过程互逆) 的执行次数相等

总结

最关键的问题莫过于 dp 数组定义的确定.

当我们把 dp 的定义设置为仅与 range(it, len) 相关, 或仅与 subarrs(it, endwith:false) or subseqs(it, endwith:false) 时, 就相当于是在解原问题的子问题 — 先解决小规模的问题, 再把规模扩大, 于是 dp 数组最后的一项就是原问题的答案.

  • dp[i][j]: 集合 subseqs(range(nums1, i), endwith:false) 和集合 subseqs(range(nums2, j), endwith:false) 中, 找相等的元素, 取最长的那一组的长度
    • 1143. Longest Common Subsequence
    • 1035. Uncrossed Lines
    • 392. Is Subsequence
  • dp[i][j]: 集合 subseqs(range(s, i), endwith:false) 中, 与 range(t, j) 相等元素的个数
    • 115. Distinct Subsequences
  • dp[i][j]: range(word1, i) 和 range(word2, j) 需要经过删除后相等的最小次数
    • 583. Delete Operation for Two Strings
  • dp[i][j]: range(word1, i) 与 range(word2, j) 需要编辑的最小次数
    • 72. Edit Distance

当我们把 dp 的定义设置为与 subarrs(it, endwith:true) or subseqs(it, endwith:true) 时, 子问题不是原问题的缩小, 原问题不是子问题的扩大, 这时的答案与 dp 数组的最后一位没有关系, 答案往往是 dp 数组的最值.

  • dp[i]: 集合 subseqs(range(nums, i+1), true) 中, 递增元素的最大长度
    • 300. Longest Increasing Subsequence
  • dp[i]: 集合 subarrs(range(nums, i+1), true) 中寻找递增元素的最大长度
    • 674. Longest Continuous Increasing Subsequence
  • dp[i]: 集合 subarrs(range(nums, i+1), true) 中, 选择和最大的元素的和
    • 53. Maximum Subarray
  • dp[i][j]: 集合 subarrs(range(nums1, i), endwith:true) 和集合 subarrs(range(nums2, j), endwith:true) 中, 找相同的元素, 取最长的那一组的长度 (最大公共后缀长度)
    • 718. Maximum Length of Repeated Subarray
