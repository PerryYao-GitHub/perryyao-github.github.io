Java 异常

异常分类

  • Throwable: 所有异常的根类

    • Error: 不用管, 处理不了

    • Exception

      • RuntimeException (UncheckedException): 运行时异常 (编译器查不出来的异常)

        • NullPointerException
        • ClassCastException: 强制转化时出现的异常
        • ArrayIndexOutOfBoundsException
        • NumberFormatException: Integer.parseInt(“123acb”);

      • CheckedException: 编译器就会查出来的异常 (通不过编译), 必须用 try catch 包围

异常的处理

使用 try … catch

声明式处理

只抛出异常, 不处理

谁调用, 谁处理 相当于抛给别人处理

包装类

八个基本数据类型 -> 包装类

boolean -> Boolean

char -> Character

byte -> Byte

short -> Short

int -> Integer

long -> Long

float -> Float

double -> Double

六个数字基本类型的包装类继承了 Number 抽象类, 它们实现了抽象方法

intValue(), longValue(), floatValue(), doubleValue(), shortValue(), byteValue()

以上是实现类型转化的, 例如 Double d = new Double(3.14); d 中有 intValue() 方法, 可以转为 int

容器 Collection

各容器关系图

List 接口 - 继承 Collection 接口 - 添加了其它方法

可重, 有序

add() | remove() | size()

  • ArrayList 类 - 实现 List 接口

    底层数组实现, 查询效率高, 增删效率低, 线程不安全

  • LinkedList 类 - 实现 List 接口

    双链表实现, 查询效率低, 增删效率高, 线程不安全

  • Vector 类 - 实现 List 接口

    底层数组, 线程安全, 效率低 (所有方法都套上了 synchronized 关键字)

    • Stack 类 - 继承 Vector 类

      是 Vector 的子类 Last In First Out

      empty(), peek(), pop(), push(), search()

Queue 接口 - 继承 Collection 接口

  • LinkedList 类 - 实现 Queue 接口

  • PriorityQueue 类 - 实现 Queue 接口

Set 接口 - 继承 Collection 接口 - 未添加其它方法

不可重复, 无序

  • HashSet 类 - 实现 Set 接口 - 底层 HashMap

    Hash 算法 (散列算法) mod n 运算

    Set 中无 index, 所以 无 get()

    add() | remove() | contains()

    HashSet 底层是 HashMap, HashMap 底层使用 数组 和 链表 实现存储

    当两个元素 Hash 值得到位置相同时, 会 调用元素 equals() 方法, 如果相同则跳过, 不同则加到链表

    数组默认长度是 16, 数组中存储的是单链表: NodeList<Object>[] hashArr = new NodeList<>[16];

    使用 HashSet 存储自定义对象

  • TreeSet 类 - 实现 Set 接口 - 底层 TreeMap

    允许对元素排序 要给定排序规则

    通过元素自身实现比较规则 (Integer, String) compareTo() 方法

    通过比较器指定比较规则

Map 接口 - 不是 Collection 的子接口

Map 接口要求双例集合 - Collection 接口要求单例集合

put(K key, V value) | remove(K key) | get(K key) | containsKey() | containsValue() | keySet() | Set<Map.Entry<K, V>> entrySet()

  • HashMap 类 - 实现 Map 接口

  • TreeMap 类 - 实现 Map 接口

    可以对键排序, 要给定Key对象的排序规则
