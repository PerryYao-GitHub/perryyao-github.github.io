use hexo step by step

全局安装 hexo-cli

cnpm install -g hexo-cli

初始化博客

hexo init

默认主题 landscape

启动

hexo s

新建一篇文章

hexo n "use hexo step by step"

新建完成后

hexo clean

hexo g

再重新启动检验
