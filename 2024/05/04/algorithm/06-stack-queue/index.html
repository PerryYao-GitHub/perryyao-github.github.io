Deque Interface & List Interface

Queue

Queue Interface

:star:继承关系

:star:主要方法

:star:主要实现类

实现类 描述 特点
LinkedList 基于链表的双端队列实现，既可以用作队列也可以用作栈。 非线程安全，性能较好
ArrayDeque 基于动态数组的双端队列实现。 非线程安全，性能更优于 LinkedList
PriorityQueue 基于堆的优先级队列。 非线程安全，元素按自然顺序或自定义比较器排序
