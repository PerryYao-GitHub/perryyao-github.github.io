Computational Complexity Analysis

Big-O Notation: upper bound of the complexity in the worst case

Static and Dynamic Arrays

Complexity

Static Array:

  • access: O(1)
  • search: O(n)
  • insertion: null
  • appending: null
  • deletion: null

Dynamic Array:

  • access: O(1)
  • search: O(n)
  • insertion: O(n)
  • appending: O(1)
  • deletion: O(n)

Singly and Doubly Linked Lists

Complexity

Singly Linked List:

  • search: O(n)
  • insert at head: O(1)
  • insert at tail: O(1)
  • remove at head: O(1)
  • remove at tail: O(n)
  • remove in middle: O(n)

Doubly Linked List:

  • search: O(n)
  • insert at head: O(1)
  • insert at tail: O(1)
  • remove at head: O(1)
  • remove at tail: O(1)
  • remove in middle: O(n)

Stack

push / pop / top (peek)

Complexity

pushing: O(1)

popping: O(1)

peeking: O(1)

searching: O(n)

size: O(1)

Queues

enqueue = add = offer / dequeue = remove = poll
